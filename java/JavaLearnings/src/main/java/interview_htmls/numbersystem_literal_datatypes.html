// binary base 2 , allowed 0,1, starts with 0B or 0b

// octal base 8 , allowed 0-7, starts with 0

// hexadecimal base 16 allowed 0-9, ABCDEF. , start with 0x 0X

// in expression, always decimal will be used, so all the values will be converted to decimal only

Indetifiers



// identifiers, name given for any data member, class, interface, enum, abstract class, method is identifier

// allowed are a-z, 0-9, $, _

// can not start with numbers

// no limit on size or use of alphabets, $, _

// can not be same as keyword of java

// String is not a keyword, it's a predefined class, so class, interface names can be same as variable name, but not recommended

For class and interfaces, the first letter has to be uppercase.Class names should be nouns

Interface names should be adjectives.

For method , variable, package_name, and constants, the first letter has to be lowercase.Methods should be verbs

One-character variable names should be avoided except for temporary variables

constant Should be all uppercase with words separated by underscores (“_”).

The prefix of a unique package name is always written in all-lowercase ASCII letters
