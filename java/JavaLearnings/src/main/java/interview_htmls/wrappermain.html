definition


A Wrapper class in Java is a class whose object wraps or contains primitive data types.

wrapper classes are used for autoboxing and unboxing, converting from primitive to object is autoboxing

and converting from object to primitive is autounboxing

there are several things which you can do only on an object and not an primitive

e.g. creating arraylist with generics, can not be done for primitive but can be done for object

so like this there are a several advantages of objects over primitive, hence wrapper classes are created to convert

primitives to object as we have a lot of utilitiy methods in wrapper classes to use

Autoboxing



// converting primitive to object is autoboxing

// Serialization can be done only for object
// as java uses only call by value, but on object a reference is just a copy of same object // so to update the passed object we have to use object rather then primitive

Wrapper Classes


following are wrapper classes for corrseponding primitives (all extends Number which is an abstract class)
int - Integer
char - Character
byte - Byte
short - Short
long - Long
double - Double
float - Float
boolean Boolean
These wrapper classes have no relation with each other, so can not be converted from one to another

Wrapper class methods

each wrapper class has below method types:

parsetXXX() // XXX is int, float, double anything, to pass string and get its primitive values

xxxValue() // to pass wrapper object and get its primitives

valueOf() // to pass primitive and get its wrapper object

toString() // to pass string and get its primitive

Boolean Wrapper


// true case insenstive is true, everything else is false

// from primitive to object {

Boolean b = Boolean.parseBoolean("true"); // true

System.out.println(b);

Boolean b1 = Boolean.parseBoolean("True"); // true, as case insensitive

System.out.println(b1);

Boolean b4 = Boolean.parseBoolean("false"); //false

System.out.println(b4);

Boolean b2 = Boolean.parseBoolean("a"); // false

System.out.println(b2);

Boolean b3 = Boolean.parseBoolean("null"); //false

System.out.println(b3);


// from object to primitive

boolean b5 = b3.booleanValue();


// from primitive to string

// only true and false case sensitive are boolean values

String s = Boolean.toString(true); // will hold true

String s1 = Boolean.toString(false); // will hold false
